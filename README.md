## Chrome

This extension allows you to control the following actions with the 3dRudder

### Video Control
* Play video
* Pause video
* Forward video
* Rewind video
* Set Volume +/-

### Player compatible
* YouTube
* Twitch
* Prime Video
* Vimeo
* Daylymotion
* ...
